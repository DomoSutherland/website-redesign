## Project Description
This project is a redesign of the Goodreads website, created as part of a web development assignment.  
The site focuses on browsing books, tracking reading activity, and encouraging users to sign up.

## Client & Purpose
**Client:** Goodreads (concept redesign) 
 Goodreads is a social reading and book discovery platform that allows users to track reading activity, discover new books, and engage with a community through reviews, ratings, and discussions.

**Purpose:**  
- Create a multi-page website
- Improve usability and visual hierarchy
- Design a functional signup form

## Design Changes from Assignment 1
Compared to Assignment 1, the following changes were made:
- Simplified the header search to reduce competition with the hero section
- Improved spacing and typography for readability
- Added a structured footer with navigation, app links, and social icons
- Added additional pages (Browse, Sign Up) to support navigation flow
- Adjusted color usage to better match the Goodreads brand
